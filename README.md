# GENERATIVE-AI-DATA-SCIENCE

This repository explores **Generative AI techniques applied to data science and analytics workflows**.

## Overview
The focus is on using large language models and generative systems to augment:
- Analysis
- Insight generation
- Reporting
- Decision support

## Focus Areas
- LLM-assisted analytics
- Prompt engineering for data workflows
- Text-to-insight pipelines
- AI-assisted reporting

## Tech Stack

- Pandas / NumPy
- LLM APIs
- Jupyter Notebooks
